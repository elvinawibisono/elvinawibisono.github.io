Weather Website

Technologies: Python, flask, javascript(jquery)

About:

A web project from Developing Acessibility UI Design. This project I coded from scratch with a priority of implementing accessibility features into the website (for example: text-to-speech, image/text magnification, etc.). The website retrieves the weather from a specified location the user provides and then projected the current weather of the location, including necessary weather information such as precipitation, humidity, and etc.

This website help provide suggestions to the user, for example in the next 3-4hrs it's going to rain , thus it suggest to bring an umbrella, or wear UV protection depending on the UV levels of that day.
